# Make The Site Public + Searchable In Google

## What is already prepared
- `robots.txt` added
- `sitemap.xml` added
- SEO meta tags (title, description, Open Graph, Twitter, canonical)
- Google-readable Organization structured data (JSON-LD)

## You still need to do (required)
1. Buy or use a domain (for example: `yourdomain.org`)
2. Host the site publicly (GitHub Pages, Netlify, Vercel, Cloudflare Pages, etc.)
3. Confirm the production domain is set in:
   - `/Users/ssgyin/Documents/New project/tiantai-lotus-website/index.html`
   - `/Users/ssgyin/Documents/New project/tiantai-lotus-website/robots.txt`
   - `/Users/ssgyin/Documents/New project/tiantai-lotus-website/sitemap.xml`
4. Open the live site and confirm it loads from the public URL
5. Submit the domain in Google Search Console
6. Submit the sitemap URL in Google Search Console:
   - `https://tiantai-lotus-sutra-association.com/sitemap.xml`

## Fastest hosting options (static site)
- GitHub Pages (free)
- Netlify (free tier)
- Vercel (free tier)
- Cloudflare Pages (free tier)

## Google Search Console steps
1. Go to Google Search Console
2. Add your property (domain or URL prefix)
3. Verify ownership (DNS TXT record is the most common method)
4. Submit sitemap
5. Request indexing for your homepage

## Important note
Google indexing is not instant. After submission, indexing can take days to weeks.
